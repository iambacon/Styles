--- layout: layout title: Header description: Component representing a page header. Contains a logo and navigation links. ---

Purpose

The header component identifies the site and provides primary navigation links.

Example

I am Bacon

Colin Bacon, web developer.

Site navigation

  • Blog
  • About
{% highlight html %}

{% endhighlight %}

Usage notes

  • Use the header once per page, before the main content.
  • Use .c-header-logo for the site identity link.
  • Use .c-header-nav-link for each primary navigation item.

Accessibility notes

  • Label the navigation so it can be distinguished from other nav regions.
  • Keep the site identity link first in the header.
  • Use visible link text for primary navigation items.
