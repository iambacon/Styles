Purpose

The aside nav groups secondary navigation links, such as categories or tags, alongside a page list.

Usage notes

  • Use .c-asideNav for secondary link groups outside the main content flow.
  • Use an ordered list when the item order is meaningful, such as most-used categories.
  • Use .c-asideNav-item-count for supporting counts.

Accessibility notes

  • Give the aside a label using a visible heading and aria-labelledby.
  • Keep counts next to their label so screen readers announce the full context.
  • Do not use counts as the only indication of importance.
