Usage notes

  • Use .c-btn as the base class for button-like links.
  • Add .c-btn--tag for the tag button treatment.
  • Add .c-btn--full when the button should fill its container.

Accessibility notes

  • Use an anchor only when the control navigates to another page or section.
  • Use a real <button> for actions that change state, submit forms or open interfaces.
  • Write link text that makes sense out of context.
