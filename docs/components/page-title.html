Purpose

The page title component introduces major pages with a prominent headline block.

Example

Recent posts for Category 'ASP.NET'

{% highlight html %}

Recent posts for Category 'ASP.NET'

{% endhighlight %}

Usage notes

  • Use .c-pageTitle to wrap the page title block.
  • Use .c-pageTitle-headline on the page heading text.
  • Add .c-pageTitle-headline--sm for long titles that need the smaller headline scale.
  • Add .has-triangle to the inner wrapper when the title needs the decorative pointer.

Accessibility notes

  • Use one <h1> for the visible page title.
  • Keep the title text descriptive because it anchors the page outline.
  • The triangle is decorative and should not carry meaning on its own.
