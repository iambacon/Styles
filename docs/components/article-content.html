Usage notes

  • Use .c-articleContent around long-form post body content.
  • Keep component-specific typography inside this wrapper so base element styles stay contained.
  • Use semantic HTML first; the component should style normal headings, tables and lists.

Accessibility notes

  • Give data tables a caption and header cells with scope.
  • Provide alt text for meaningful inline images.
  • Use heading levels in document order so the article body remains easy to navigate.
