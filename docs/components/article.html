Usage notes

  • Use .c-article for post cards and listing summaries.
  • Use .c-article-title for the linked post title.
  • Use .c-article-meta for categories, tags and other supporting metadata.
  • Keep the title link block-level so the two-line truncation behaves as expected.

Accessibility notes

  • Use meaningful image alt text when the image link is part of the article teaser.
  • Keep metadata labels available with visually hidden <dt> text.
  • Make the image and title links point to the same article destination in production.
