Purpose

Tags label article metadata and help readers move to related posts.

Example

ASP.NET
{% highlight html %} ASP.NET {% endhighlight %}

Usage notes

  • Use .c-tag for compact metadata labels.
  • Add .c-tag--category when the tag represents an article category.
  • Keep tag text short enough to scan in article lists and side navigation.

Accessibility notes

  • Use a link when the tag navigates to filtered results.
  • Use plain text instead of a link when the tag is only descriptive metadata.
  • Avoid relying on the background colour alone to communicate meaning.
