Usage notes

  • Use .c-linksList for grouped footer link columns.
  • Use .c-linksList-items--posts for ordered article lists.
  • Use .c-linksList-items--social for the social profile group.
  • A .c-linksList-link is designed for two lines of text; longer titles are clipped to preserve the footer layout.

Accessibility notes

  • Use headings to label each group of links.
  • Keep each link destination unique in production, even when examples use local anchors.
  • Keep social link names available to screen readers when using hidden-text icon styles.
