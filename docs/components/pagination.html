Purpose

Pagination helps readers move through a long list of posts without losing their place.

Example

1 ... 4 5 6

Usage notes

  • Use .c-pagination-page for individual page links.
  • Use .c-pagination-page--current for the current page.
  • Use .c-pagination-more for the larger load-more action.

Accessibility notes

  • Add an accessible label to the pagination navigation.
  • Mark the current page with aria-current="page".
  • Use plain text for ellipses because they are not interactive.
