Colour utilities apply one-off background or text colours without creating a new component class.

Usage notes

  • Use these utilities for small composition adjustments, not as a substitute for component styling.
  • Prefer component variables when a colour belongs to a reusable pattern.

Accessibility notes

  • Check text contrast any time a background colour utility is paired with text.
  • Do not use utility colours as the only indication of meaning.
