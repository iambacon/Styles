Usage notes

  • Use layout containers to control measure and page rhythm.
  • Use .layoutContainer--visible only when a child element intentionally escapes the container.
  • Prefer semantic wrappers and add grid utilities as layout helpers.

Accessibility notes

  • Keep source order meaningful when layout utilities change visual placement.
  • Test long content inside constrained containers at mobile widths.
