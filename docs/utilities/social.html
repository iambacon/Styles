Purpose

Social utilities apply sprite positions for supported social profile icons.

Examples

LinkedIn Twitter RSS Google
{% highlight html %} LinkedIn Twitter RSS Google {% endhighlight %}

Usage notes

  • Use .social with one service class: .linkedIn, .twitter, .rss or .google.
  • Pair icon links with .hideText when the text is visually replaced by the sprite.

Accessibility notes

  • Keep readable link text in the anchor so screen readers get a useful name.
  • Use real destinations in production rather than example anchors.
