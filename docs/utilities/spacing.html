--- layout: layout title: Spacing description: Margin and padding helper classes. Based on line height for vertical rhythm. ---

Purpose

Spacing utilities apply small margin and padding adjustments based on the base line-height rhythm.

Reference

Class Description Spacing
.m-r--none Margin right 0px
.m-r--half Margin right 12px
.m-r--full Margin right 24px
.m-l--none Margin left 0px
.m-b--full Margin bottom 24px
.m-b--none Margin bottom 0px
.m-br--quarter Margin bottom and right 6px
.m-t--full Margin top 24px
.m-tb--full Margin top and bottom 24px
.p-t--half Padding top 12px
.p-t--full Padding top 24px
.p-b--3 Padding bottom 72px

Usage notes

  • Use spacing utilities for small one-off layout adjustments.
  • Prefer component spacing when the spacing is part of a reusable pattern.
  • Keep new spacing helpers tied to the base vertical rhythm.

Accessibility notes

  • Do not use spacing utilities to create visual order that differs from source order.
  • Check compact layouts with zoomed text and long labels.
