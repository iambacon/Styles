Styles is a pattern library for iambacon websites. It provides guidelines, reusable UI components and utilities to ensure a consistent experience and help simplify the developer workflow.

Styles is still a work in progress. It is currently used for www.iambacon.co.uk and this documentation website itself. The project is available on GitHub for you to follow, fork or contribute.

Help

Need help using Styles? Please file an issue on GitHub.
