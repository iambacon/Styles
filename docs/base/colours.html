Usage notes

  • Use semantic colour variables through base and component Sass modules.
  • Use utility colour classes only when a one-off background or text colour is required.
  • Prefer named variables to raw hex values in source styles.

Accessibility notes

  • Check contrast before pairing palette colours with text.
  • Do not use colour alone to communicate state or category.
