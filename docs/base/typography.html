Purpose

Typography variables define the base font stack, scale and rhythm used throughout the site styles.

Variables

Name Description Value
$base-font-size The base font size. 16px
$base-line-height The base line height determines the vertical rhythm unit. 24px
$base-font-family The default font family. Lato, Trebuchet MS, sans-serif
$h1-font-size The font size for h1. 60px
$h2-font-size The font size for h2. 46px
$h3-font-size The font size for h3. 24px
$h4-font-size The font size for h4. 16px
$h5-font-size The font size for h5. 14px
$h6-font-size The font size for h6. 11px

Usage notes

  • Use the typography variables through mixins and utility classes.
  • Keep vertical spacing tied to $base-line-height.
  • Use heading utility classes only when the visual scale needs to differ from the semantic heading level.

Accessibility notes

  • Keep heading levels in document order even when using visual heading classes.
  • Check line lengths and wrapping at mobile widths.
