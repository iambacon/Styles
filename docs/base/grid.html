Purpose

Grid variables define the breakpoints and spacing used by layout mixins and utilities.

Variables

Name Value Form factor
$screen-sm-min 768px Tablet and up
$screen-md-min 992px Desktop and up
$screen-lg-min 1200px Large desktop and up
$screen-xs-max 767px Phone
$screen-sm-max 991px Tablet
$screen-md-max 1199px Desktop
$gutter-width 24px Base line height
$margin-width 20px Page edge spacing

Usage notes

  • Extra-small screens are the default; larger breakpoints are layered on top.
  • Use the grid variables through mixins rather than hard-coding breakpoint values.
  • Keep layout utility examples aligned with these breakpoint names.

Accessibility notes

  • Check layouts at each breakpoint with long headings and enlarged text.
  • Do not rely on viewport width alone to hide essential content.
