Purpose

Code styles make inline and block examples readable inside posts and documentation.

Example

public class RaygunLoggerProvider : ILoggerProvider
{
    private readonly RaygunConfiguration _config;

    public ILogger CreateLogger(string categoryName)
    {
        return new RaygunLogger(categoryName, _config);
    }
}
{% highlight html %}

Usage notes

  • Use <code> for inline code references.
  • Wrap longer examples in <pre><code> so spacing is preserved.
  • Keep code examples short enough to scan in article content.

Accessibility notes

  • Do not use screenshots of code when text code will work.
  • Keep line lengths reasonable so code does not force horizontal scrolling on small screens.
